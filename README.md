# Katar Population Prediction

Dieses Projekt zeigt eine **Vorhersage der Bevölkerung von Katar** mithilfe einer **linearen Regression**. Historische Bevölkerungsdaten werden analysiert, und zukünftige Werte werden prognostiziert. Die Visualisierung erfolgt mit `matplotlib` und `seaborn`.  

> Dieses Projekt diente mir als **Einstieg in Machine Learning**, um den Umgang mit Daten, linearen Modellen und Visualisierungen zu lernen.  

## Features

- Analyse historischer Bevölkerungsdaten von Katar  
- Vorhersage zukünftiger Bevölkerungszahlen mit linearer Regression  
- Datenvisualisierung mit `matplotlib` und `seaborn`  
- Einfacher, verständlicher Code für Einsteiger in Data Science
